# UsagiLab 兔兔实验室

## Projects 项目

- [UsagiPass](https://dxpass.turou.fun/): Dynamically Generate DXPASS Style Wahlap QR Code. 动态生成 DXPASS 风格的玩家二维码。
- [UsagiCard](https://uc.turou.fun/): Customize your own DXPASS-style NFC card. 定制专属DXPASS风格NFC小卡片。
- [maimai.py](https://maimai.turou.fun/): The definitive python wrapper for MaimaiCN related development. 用于国服舞萌相关开发的最佳Python工具库。
